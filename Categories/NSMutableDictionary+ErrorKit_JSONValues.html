Instance Methods

setArray:forKey:withError:

If anArray is an NSArray, the given key-value pair is added to the dictionary.

- (BOOL)setArray:(NSArray *)anArray forKey:(id<NSCopying>)aKey withError:(NSError **)errorPtr

Parameters

anArray

The value for aKey. anArray may be nil, but then the key-value pair is not added to the dictionary.

aKey

The key for value. aKey may be nil nil, but then the key-value pair is not added to the dictionary.

errorPtr

A pointer to an NSError object or NULL. You do not need to create an NSError object.

Return Value

YES if the key-value pair was added, NO otherwise.

Discussion

If anArray is an NSArray, the given key-value pair is added to the dictionary.

If aKey and/or anArray are nil, no NSInvalidArgumentException is raised, but errorPtr will contain an NSError object.
